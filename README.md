# 🎓 Courses Selling App (MERN Stack)

A full-stack **Courses Selling Web Application** built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js).  
This platform allows admins to create and manage courses, while users can browse and purchase courses seamlessly.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Admin authentication (JWT based)
- Create, update, and delete courses
- Upload course images
- Manage course pricing and descriptions

### 👤 User Side
- User registration & login
- View available courses
- Course details page
- Secure API communication

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- shadcn/ui
- Axios
- Context API (State Management)

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Express file uploading (Image Uploads)
